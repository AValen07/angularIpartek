Zona privada

solo pueden acceder usuarios logueados

Para proteger el acceso a esta RUTA debemos usar una GUARD y definirla en el routing

path: 'privado', component: PrivadoComponent, canActivate: [LoginGuard]

canActivate
Guarda para indicar si se puede actvar esa ruta
LoginGuard
Guarda creada por nosotros que retornara true o false
